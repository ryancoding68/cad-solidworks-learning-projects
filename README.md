# cad-solidworks-learning-projects
Repository ini berisi projek desain CAD menggunakan SolidWorks yang dilengkapi video tutorial dan file desain (part, assembly, dan drawing). Cocok untuk pemula dan mahasiswa sebagai media belajar, latihan, dan portofolio. Seluruh materi disusun rapi dan dapat diakses melalui GitHub untuk tujuan edukasi.
